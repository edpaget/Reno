## Reno

The Backbone + Bootstrap frontend for [edpaget/Reno-api](https://github.com/edpaget/Reno-api).

### Usage

#### Dependencies
Reno requires [Brunch](https://github.com/brunch/brunch) to build.

#### Installation
```
git clone https://github.com/edpaget/Reno
cd Reno
npm install .
./build.sh
```
